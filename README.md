# alice-chiko.github.io
=======

в рамках 1 лаборторной работы было создано оформление сайта, через промпт в нейросети. я старалась сделать оформление максимально приятным, поэтому оно выполнено в нежных пастельных тонах. цвет текста сочетается с общей цветовой гаммой. основная часть страницы выполнена в практически белом цвете, для лучшей читабельности. также по моему желанию были добавлены фотографии, которые отлично дополняют портфолио, и анимации, которые делают восприятие сайта более приятным. текст я предпочла разместить по всей площади страницы, а не только в выделенном изначально кусочке.
