###  Objective
Целью работы является разработка программы на языке Python для генерации бинарного дерева заданной высоты с использованием рекурсивного подхода. Основное внимание уделено работе с вложенными структурами данных (словари), реализации кастомных правил ветвления и различным способам визуализации иерархии (JSON, уровни, текстовая графика).

### Implementation
В ходе работы был реализован модуль для генерации и отображения дерева:

1.  **`gen_bin_tree`**: Основная рекурсивная функция. Она строит дерево сверху вниз. Каждый узел — это словарь с ключами `root`, `left` и `right`. Высота дерева контролируется базовым случаем рекурсии (`height == 0`).
2.  **Функции ветвления (`left_branch`, `right_branch`)**: Определяют математическую логику вычисления значений дочерних узлов (в данном случае: $root + 2$ для левого и $root * 3$ для правого).
3.  **Класс `BinaryTreeVisualizer`**: Позволяет группировать узлы по уровням, обеспечивая удобный анализ структуры дерева.
4.  **Визуализация**: Реализованы функции для вывода дерева в формате JSON и в виде горизонтального графа с отступами.

### Testing
Для проверки корректности построения дерева реализован класс `TestBinaryTree`, включающий **6 тестовых сценариев**:
* Проверка базовых случаев: дерево высотой 0 (возвращает `None`) и высотой 1 (только корень).
* Проверка правильности вычислений значений в узлах для высоты 2 и 3.
* Тестирование краевых значений: работа с отрицательным корнем и нулевым значением.
